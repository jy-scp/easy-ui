# Modal

Headless Modal 组件，只提供逻辑，样式完全由你控制。

## 预览

## 基本用法

```tsx
import { Modal, useModalContext } from '@easy/ui'

const Trigger = () => {
  const { setIsOpen } = useModalContext()
  return <button onClick={() => setIsOpen(true)}>打开</button>
}

const Content = () => {
  const { setIsOpen } = useModalContext()
  return (
    <div>
      <p>Modal 内容</p>
      <button onClick={() => setIsOpen(false)}>关闭</button>
    </div>
  )
}

export default function App() {
  return (
    <Modal>
      <Trigger />
      <Modal.Portal>
        <Modal.Overlay className="fixed inset-0 bg-black/50" />
        <Modal.Content className="fixed left-1/2 top-1/2 -translate-x-1/2 -translate-y-1/2 bg-white p-6 rounded-lg">
          <Content />
        </Modal.Content>
      </Modal.Portal>
    </Modal>
  )
}
```

## API

### `<Modal>`

| 属性 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `defaultOpen` | `boolean` | `false` | 初始是否打开 |
| `children` | `ReactNode` | - | 子节点 |

### `<Modal.Portal>`

将内容渲染到 `document.body`，避免层叠上下文问题。

### `<Modal.Overlay>`

遮罩层，接受所有 `div` 属性。

### `<Modal.Content>`

对话框容器，支持 ESC 关闭，接受所有 `div` 属性。

### `useModalContext`

在 `<Modal>` 子树内使用，获取 `isOpen` 和 `setIsOpen`。
